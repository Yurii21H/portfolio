# 🌐 Personal Portfolio Website

A modern, dark-themed developer portfolio website built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean, fast, handcrafted code.

## ✨ Features

- Fully responsive — works on mobile, tablet, and desktop
- Smooth scroll animations (Intersection Observer API)
- Dark theme with animated gradient glow effects
- Noise texture overlay for depth
- Sections: Hero, About, Skills, Projects, Contact
- Accessible and SEO-friendly markup

## 🚀 Getting Started

No installation required. Just open the file in your browser:

```bash
# Clone the repository
git clone https://github.com/yourusername/portfolio.git

# Open in browser
open index.html
```

Or simply double-click `index.html`.

## 📁 Project Structure

```
portfolio/
└── index.html       # Complete single-file app (HTML + CSS + JS)
```

## 🎨 Customization

To make it your own, edit `index.html` and update:

| Section | What to change |
|---------|---------------|
| Hero | Your name, title, and tagline |
| About | Your bio and stats |
| Skills | Your actual tech stack |
| Projects | Your real project cards |
| Contact | Your email and social links |
